Trabajo #1 — Adición

Problema:
---------

Escribir los pasos para mostrar la suma de dos números que ingresa el usuario.

Entregables:
------------

- Sufijo de carpeta: Adición
- readme.md
- Implementación: Adición.cpp.


Análisis:
-----------

Dados dos números enteros, se realiza y se muestra la sumatoria entre los mismos, siguiendo el siguiente procedimientos: 

1. Se ingresan los números enteros "A" y "B".
2. Se realiza la sumatoria entre "A" y "B".
3. Se muestra el resultado de dicha sumatoria, identificada como "C".

![alt tag](http://image.prntscr.com/image/d65e5d75e297456ea8090ba1bd902d75.png)


Algoritmo:
----------

1. Ingresar A
2. Ingresar B
3. Leer A
4. Leer B
5. Sumar A+B
6. Mostrar C
